R2021a激活流程

拷贝文件夹Matlab910Win中的libmwlmgrimpl.dll文件到R2021a安装位置下的路径"\bin\win64\matlab_startup_plugins\lmgrimpl"中;

拷贝文件夹Matlab910Win中的license.lic文件到R2021a安装位置下的路径“\licenses”中。

安装完后，发现开始菜单的图标不能正确显示。

按照以下操作自行创建开始菜单图标即可。
